# Overview
Generic implementation of model railway block occupation using positional train detection at each end of block. Also cascading multi aspect colour light signalling written in Microchip MPASM assembly languge.

Needs to be included by a specialisation that provides aspect display appropriate for a particular multi aspect colour light signalling regime.
# Dependency
Requires [utility_pic](https://github.com/MonitorComputing/utility_pic).
